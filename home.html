Welcome to OpenAlt! On this website, you can help add alt-text to images on the web that don't already have them. In addition, you can edit alt-texts that others have made to ensure that they follow the guidlines for descriptive alt-texts below. These alt-texts will be used by screen readers to describe images to people who are blind. When writing alt-texts, they should follow the guidlines below.


Guidelines for Adding Alt Text
General:

  • Be succinct while still maintaining key elements of the image being described
  • Remove filler or unimportant words when possible
  • Describe the information, not the picture
  • Often times, it is more important to describe the significance of the image rather than the shape, i.e. Twitter's logo should be described as "Twitter" in alt text and not "blue bird"
Specific:
For decorative images:
- Leave the alt text blank
For informative images1:
Complex images (such as charts or graphs):
- Include a general statement describing what the image outlines
- Include a summary of the general trend of the data
- Include a link to a long description containing specific numerical values shown in tables and charts
Textual Images (such as landscapes):
- Include relevant and important details
- Be descriptive while remaining concise
For active images (such as buttons or links):
- Include what the image does when clicked (i.e. where a link would send the user)
1To determine whether an image is decorative or informative:
If removing the image detracts from the user's understanding, it is informative. If not, it is decorative.
